# Changelog

All notable changes to the **TV Remote for Android TV** project will be documented in this file.

## [1.4.0] - 2026-06-25

### Added
- **Initial Public Open-Source Release**: Main repository created for community contributions.
- **Apache License 2.0**: Added open-source license.
- **Project Documentation**: Added comprehensive `README.md`, `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`, `SECURITY.md`, and CI build scripts.

### Changed
- **100% Free & Ad-Free Model**: Removed all monetization, AdMob SDK, billing client integrations, paywalls, and "Go Premium" options.
- **Performance Optimizations**: Simplified app splash screen startup time, removed unused dependencies, and cleaned up background work.
- **Refactoring & Reorganization**: Cleaned up the codebase structure for easier maintenance.
